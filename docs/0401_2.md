# [UniNote] 2026-04-01 작업 일지 (2)

## 1. 작업 개요
- **목표**: 로그인 성공 후 학생의 수강 강의 목록을 보여주는 대시보드 기능 구현
- **진범위**: 백엔드 API 개발, 프론트엔드 화면 구현 및 연동 (CORS 설정 포함)

---

## 2. 백엔드 (Spring Boot) 상세 작업

### 2.1. 데이터 접근 계층 (Repository)
- `CourseRepository`: 강의 정보 조회를 위한 기본 JPA 인터페이스 생성
- `EnrollmentRepository`: 학생별 수강 내역 조회를 위한 `findByStudent` 메서드 추가

### 2.2. 서비스 및 컨트롤러 (Service & Controller)
- `DashboardService`: 
    - 학생 정보를 확인하고 해당 학생의 모든 `Enrollment` 정보를 조회
    - `CourseResponse` DTO로 변환하여 반환하는 로직 구현
- `DashboardController`: 
    - `GET /api/dashboard/courses` 엔드포인트 생성
    - JWT 토큰에서 추출한 `Authentication` 정보를 통해 현재 로그인한 학생의 데이터를 반환

### 2.3. 보안 설정 (SecurityConfig)
- **CORS 설정 추가**: 프론트엔드(`http://localhost:5173`)로부터의 요청을 허용하도록 `CorsConfigurationSource` 설정
- **권한 설정**: 대시보드 관련 API는 인증된 사용자만 접근 가능하도록 설정

---

## 3. 프론트엔드 (React + Vite) 상세 작업

### 3.1. API 통신 모듈 (api/client.js)
- `axios` 인스턴스 생성 및 인터셉터 설정
- **요청 인터셉터**: 모든 요청 헤더에 `localStorage`의 JWT를 `Authorization: Bearer {token}` 형식으로 자동 첨부
- **응답 인터셉터**: 401(Unauthorized) 에러 발생 시 토큰을 제거하고 로그인 페이지로 자동 리다이렉트

### 3.2. 페이지 및 컴포넌트 구현
- **LoginPage.jsx**: 
    - 학번/비밀번호 입력 폼 및 백엔드 `/api/auth/login` 연동
    - 로그인 성공 시 토큰 저장 및 대시보드 이동 로직 구현
- **DashboardPage.jsx**: 
    - 컴포넌트 마운트 시 `/api/dashboard/courses` API 호출
    - Tailwind CSS를 활용하여 수강 중인 강의 목록을 카드 형태의 UI로 출력
- **App.jsx (Routing)**:
    - `react-router-dom`을 사용하여 `/login`, `/dashboard` 경로 정의
    - 토큰 존재 여부에 따른 조건부 렌더링(보호된 라우트) 적용
