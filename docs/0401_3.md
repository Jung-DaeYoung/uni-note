# [UniNote] 2026-04-01 작업 일지 (3)

## 1. 작업 개요
- **목표**: 실제 데이터를 활용한 기능 검증 및 대시보드 UI/UX 고도화
- **주요 성과**: 비밀번호 검증 시스템 구축, 대형 화면용 대시보드 레이아웃 완성

---

## 2. 백엔드 및 DB 고도화

### 2.1. 비밀번호 보안 시스템
- **Entity 수정**: `Student` 클래스에 `password` 필드 추가 및 DB 컬럼 반영 (`ALTER TABLE`).
- **Auth 로직 업데이트**: `AuthService`에서 입력받은 비밀번호를 DB와 대조하는 로직 구현.
    - *참고*: 현재 테스트 편의를 위해 `BCrypt` 대신 평문(String `equals`) 비교 방식을 임시 적용 중.
- **Security 설정**: `PasswordEncoder` 빈 등록 및 프론트엔드 연동을 위한 CORS 설정 보완.

### 2.2. 테스트 데이터 세팅 (DBeaver)
- **통합 SQL 스크립트 작성**: 
    - 교수(5명), 학생(1명 - 본인), 강의(6개), 수강신청 데이터 일괄 삽입.
    - `TRUNCATE`를 포함하여 실행 시마다 데이터를 초기화할 수 있도록 구성.

---

## 3. 프론트엔드 UI 리뉴얼 (Vite + React)

### 3.1. 글로벌 스타일 및 테마
- **Theme**: 화이트 배경과 블루 포인트 컬러(`#2563eb`)를 사용한 모던 대학 포털 디자인.
- **Font**: Pretendard 등 시스템 폰트를 활용하여 가독성 확보.

### 3.2. 대시보드 UI 최적화
- **사이드바(Sidebar)**: 다크 톤의 좌측 메뉴바를 도입하고, 현재 활성화된 '내 강의 대시보드' 항목만 남겨 심플하게 구성.
- **컴팩트 강의 카드(Course Cards)**:
    - 하단 히트맵/최신글 영역 확보를 위해 카드 크기 축소.
    - 강의 코드(예: MATH101)를 제거하고 강의명과 교수명 위주로 깔끔하게 배치.
    - 마우스 오버 시 상단 포인트 라인 및 부드러운 애니메이션 효과 추가.
