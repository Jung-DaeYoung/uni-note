# 프로젝트 작업 일지 (2026-04-09)

**주요 성과**: 프론트엔드 UI 현대화(Tailwind CSS 도입), 3단 가변 레이아웃 구현, 백엔드 노트/게시판 도메인 및 API 완성.

---

## 1. 프론트엔드 환경 구축 (Frontend Infrastructure)

### 1.1. Tailwind CSS v4 도입 및 설정
- **패키지 설치**: `tailwindcss`와 Vite용 공식 플러그인 `@tailwindcss/vite` 설치.
- **Vite 설정 업데이트**: `vite.config.js`에 Tailwind 플러그인 등록 및 `@` 경로 별칭(Alias) 설정 완료.
- **CSS 연동**: `index.css`에 `@import "tailwindcss";`를 추가하여 전역 스타일링 기반 마련.
- **아이콘 라이브러리**: `lucide-react`를 설치하여 현대적이고 일관된 아이콘 시스템 구축.

### 1.2. 레거시 코드 정리
- **App.css**: 기존의 복잡한 CSS 클래스를 모두 Tailwind로 전환함에 따라 파일을 비우고 유지보수성을 높임.

---

## 2. UI/UX 디자인 및 구현 (Frontend Refactoring)

### 2.1. 로그인 페이지 (LoginPage)
- **비주얼 강화**: 다크 모드 기반의 브랜딩 패널과 라이트 모드 기반의 입력 폼이 결합된 스플릿 레이아웃 적용.
- **사용자 경험**: 로딩 상태 표시 및 입력 필드 포커스 시 애니메이션 효과 추가.

### 2.2. 대시보드 페이지 (DashboardPage)
- **카드형 인터페이스**: 수강 중인 강의를 직관적인 카드로 표현.
- **통계 요약 바**: 수강 강의 수, 전체 노트 수 등을 한눈에 확인 가능한 대시보드 상단 바 추가.
- **반응형 사이드바**: 왼쪽 내비게이션 바를 접고 펼 수 있는 토글 기능 구현.

### 2.3. 강의 상세 페이지 (CourseDetailPage) - 핵심 기능
- **3단 가변 레이아웃 (Tri-pane Layout)**: 
    1. **Left**: 글로벌 내비게이션 및 프로필 관리.
    2. **Center**: 스마트 필기 에디터 (학습 목표, 메인 필기, 질문 리스트).
    3. **Right**: 실시간 느낌의 익명 질문 게시판 사이드바.
- **스마트 기능**:
    - **자동 저장 (Auto-save)**: 사용자가 입력을 멈추면 2초 후 백엔드로 데이터를 전송하여 데이터 손실 방지.
    - **상태 표시줄**: 글자 수 계산 및 저장 상태(Synced / Saving) 실시간 표시.

## 3. 백엔드 시스템 개발 (Backend Development)

### 3.1. 데이터 도메인 확장 (Entity)
- **Note**: 강의별, 학생별 필기 데이터를 관리하는 엔티티 생성.
    - 필드: `goals`, `content`, `questions`, `aiSummary`, `aiQuizData`.
- **Post**: 익명 질문 게시판을 위한 엔티티 생성.
    - 필드: `content`, `isAnonymous`, `createdAt`.
- **JPA Auditing**: `BackendApplication`에 `@EnableJpaAuditing`을 적용하여 생성/수정 시간 자동 관리.

### 3.2. API 계층 구현
- **Note API**:
    - `GET /api/notes/{courseId}`: 기존 필기 데이터 조회.
    - `POST /api/notes/{courseId}`: 필기 데이터 자동 저장 및 업데이트.
- **Post API**:
    - `GET /api/posts/{courseId}`: 강의별 게시글 목록 최신순 조회.
    - `POST /api/posts/{courseId}`: 익명 게시글 작성.

### 3.3. DTO 및 서비스 로직
- **익명성 보장**: `PostService`에서 학생 ID를 기반으로 무작위 번호(`익명 1`, `익명 2` 등)를 부여하는 로직 구현.
- **노트 조회 최적화**: 기존 노트가 없을 경우 빈 템플릿을 자동으로 생성하여 반환하도록 설계.
