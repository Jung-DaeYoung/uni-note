# 2026-04-10 주간 작업 요약 (0410_1.md)

## 1. 프로젝트 환경 동기화
- 로컬 Git 프로젝트와 GitHub(`Jung-DaeYoung/uni-note`) 원격 저장소 완전 동기화 완료.

## 2. 프런트엔드 리팩토링
- **전역 상태 관리**: `AuthContext`를 도입하여 `localStorage` 직접 의존성 제거 및 전역 인증 상태 관리 구현.
- **레이아웃 구조화**: `AppLayout` 컴포넌트를 생성하여 사이드바와 헤더를 공통화, 코드 중복 제거 및 유지보수성 향상.
- **라우팅 최적화**: `ProtectedRoute`를 도입하여 인증이 필요한 페이지에 대한 접근 제어 추상화 및 `App.jsx` 가독성 개선.
- **코드 정리**: 중복되는 게시판 페이지(`AnonymousBoardPage`) 삭제 및 라우트 정리.

## 3. 백엔드 보안 및 예외 처리
- **전역 예외 처리**: `GlobalExceptionHandler`와 `@RestControllerAdvice`를 사용하여 표준화된 에러 응답(`ErrorResponse`) 규격 마련.
- **인가(Authorization) 보안**: `EnrollmentRepository`를 활용해 모든 리소스 조회/저장 시 수강생 검증 로직(`validateEnrollment`) 추가. 무단 접근 시 `CourseAccessException`을 던져 403 Forbidden 응답.

## 4. 게시판 기능 확장 준비
- **도메인 확장**: `Post` 엔티티에 `title` 필드 및 `Comment`와의 1:N 관계 추가.
- **데이터 모델**: `Comment` 엔티티 신규 생성 및 DTO(CommentResponse, PostResponse) 확장.
- **비즈니스 로직**: `PostService`에 댓글 작성 및 상세 조회 시 댓글 리스트 포함 로직 구현.
