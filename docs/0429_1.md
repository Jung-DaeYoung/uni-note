# 개발 일지 (2026-04-29)

## 1. 주요 작업 내용
- **에디터 아키텍처 리팩토링 (Notion 스타일)**:
  - 기존의 불안정한 슬래시 명령어 방식을 `@tiptap/suggestion` 기반으로 교체하여 안정성 확보.
  - 에디터 로직을 Extension, Component, Main Editor로 분리하여 코드 유지보수성 향상.
  - `tippy.js`를 이용한 명령어 UI 구현.
- **블록 기반 에디터 구조 도입**:
  - `BlockId` 확장(Extension)을 추가하여 모든 최상위 블록에 고유 ID(UUID)와 드래그 속성 주입.
  - `BlockHandle` 컴포넌트를 구현하여 각 블록 좌측에 드래그 핸들 및 복제/삭제 메뉴 제공.
  - 마우스 호버 시 블록 배경색 하이라이트(노션 스타일) 적용.
- **에디터 안정화**:
  - `403 Forbidden` 저장 에러 원인 파악 및 디버깅 로그 추가.
  - `tippy.js` 패키지 설치 및 의존성 이슈 해결.
  - 에디터 리팩토링 과정에서 발생한 ReferenceError 수정.

## 2. 해결된 이슈
- 에디터 임포트 경로 불일치 문제 해결.
- `tippy.js` 라이브러리 추가.
- 블록 핸들 좌표 계산 로직 수정 및 렌더링 오류 수정.
- 상태 관리 변수(isMenuOpen) 순서 이슈 해결.

## 3. 향후 계획
- 블록 단위 AI 연동 기능 구현 (요약 및 문제 생성).
