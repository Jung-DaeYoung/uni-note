## ✅ 주요 구현 사항

### 1. 블록 메뉴 UX 개선
- **기능**: 블록 핸들(`::`) 클릭 시 나타나는 복제/삭제 메뉴 외부 클릭 시 자동 닫기.
- **해결**: `useRef`와 `mousedown` 이벤트 리스너를 활용하여 메뉴 외부 영역 감지 로직 구현.

### 2. 블록 기반 아키텍처 리팩토링
- **블록 고유 ID**: 모든 노드(Paragraph, Heading 등)에 `data-id` 자동 할당 시스템 구축 (`BlockId` 확장).
- **드래그 앤 드롭 (DnD)**:
    - `BlockHandle`을 드래그하여 블록 순서 변경 기능 구현.
    - `insertContentAt` 및 `deleteRange`를 활용한 안정적인 데이터 이동.
    - 이동 방향에 따른 Index Shift 보정 로직 적용.

### 3. 슬래시 명령어 (Slash Command) 안정화
- **실행 흐름 복구**: `SuggestionList`에서 Tiptap 표준 `props.command(item)` 호출 방식으로 수정.
- **상태 초기화 로직**: 블록 변환 실패를 방지하기 위해 `setParagraph()`를 통한 샌드박스 초기화 후 `toggle` 명령어 실행.
- **커서 보정**: `setTextSelection(range.from)`을 적용하여 명령어 실행 후 커서가 튀는 현상 해결.
- **UI 개선**: 노션 스타일의 동적 명령어 리스트 및 검색 필터링 구현.

### 4. 에디터 UI/UX 디자인 (Notion 스타일)
- **코드 블록**: 연한 회색 배경, 둥근 모서리, `Fira Code` 폰트 적용.
- **인용구/제목**: 명시적 CSS를 통한 시각적 가독성 확보.
