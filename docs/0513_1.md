# 2026-05-13 작업 요약 보고서

## 1. PDF 링크 블록 구현 (PDF Link Block)
에디터 내에 PDF 파일을 첨부하고 관리할 수 있는 독립적인 블록 기능을 추가했습니다.
- **백엔드**: `/api/upload/file` 엔드포인트를 통한 PDF 업로드 로직 구현.
- **프론트엔드**: Tiptap 커스텀 `PdfBlock` 익스텐션 제작.
  - 카드 UI (아이콘, 파일명, 메타데이터).
  - '새 탭 열기' 및 '강제 다운로드' 기능 제공.
- **강제 다운로드**: 브라우저 PDF 뷰어를 우회하여 사용자 PC에 원본 파일명으로 즉시 저장되는 전용 API(`api/upload/download`) 구현.
- **사용성**: 슬래시 명령어(`/PDF 업로드`) 및 드래그 앤 드롭 연동.

## 2. AI 멀티모달 퀴즈 생성 (AI Multimodal Quiz)
기존 텍스트 기반 퀴즈 생성을 넘어, 이미지와 PDF 내용을 실제 분석하여 문제를 생성하는 기능을 구축했습니다.
- **멀티모달 통합**: Gemini 2.5 Flash 모델을 활용하여 텍스트 + 이미지 + PDF 데이터를 동시에 분석.
- **데이터 전처리**: Tiptap JSON에서 미디어 URL을 추출하고, 서버 로컬 파일을 Base64로 인코딩하여 AI에 전달하는 파이프라인 구축.
- **할당량 최적화**: 미디어 개수 제한 및 재시도 로직을 통해 API 할당량(429 Error) 관리 최적화.

## 3. 시스템 안정성 및 보안 강화
- **Jackson 파싱 오류 방지**: AI 응답 내 숫자 길이를 500자로 제한하는 프롬프트 엔지니어링을 적용하여 `getMaxNumberLength` 제한으로 인한 런타임 에러 해결.
- **데이터 정제**: 불필요한 로우 데이터나 복잡한 수식을 배제하고 텍스트 중심의 간결한 설명을 유도하도록 AI 가이드라인 설정.

## 4. 향후 계획
- 생성된 퀴즈의 출처(이미지/PDF 블록) 하이라이팅 기능 고도화.
- 대용량 PDF 처리를 위한 토큰 관리 최적화.
