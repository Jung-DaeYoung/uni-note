# 2026-05-14 작업 내역

## 1. 로그인 페이지 UI 단순화 및 개선
- **디자인 변경**: 복잡한 브랜딩 요소를 제거하고 "UniNote" 로고와 학번/비밀번호 입력란만 포함된 심플한 카드 레이아웃으로 변경.
- **배경색 수정**: 배경을 어두운 톤(`bg-slate-900`)으로 변경하여 중앙 로그인 폼의 몰입감을 높임.

## 2. 강의 내비게이션 및 사이드바 구조 개편
- **전역 상태 관리**: `CourseContext.jsx`를 생성하여 수강 중인 강의 목록을 앱 전체에서 공유.
- **계층형 사이드바**: 대시보드 메뉴 하위에 강의 목록이 나열되도록 구현.
- **내비게이션 최적화**: 
    - 강의명을 클릭하면 즉시 해당 강의의 노트 에디터로 이동하도록 동선 단축.
    - 대시보드 토글 버튼(화살표)은 페이지 이동 없이 목록만 열고 닫도록 개선.
- **에디터 연동**: `CourseDetailPage.jsx`가 전역 강의 상태를 사용하여 브레드크럼 및 내비게이션을 동적으로 처리하도록 리팩토링.

## 3. OX 퀴즈 해설 화면 오류 수정
- **문제**: OX 퀴즈 결과 화면에서 본인이 선택한 답이 표시되지 않던 현상.
- **해결**: `CBTPlayer.jsx`에서 `OX` 타입을 명시적으로 처리하도록 로직을 추가하여 정답/오답 및 사용자 선택지가 명확히 시각화되도록 수정.

## 4. 코드 블록 하이라이팅 및 언어 선택 기능 구현
- **문법 하이라이팅**: Tiptap에 `lowlight`를 연동하여 소스 코드의 문법을 자동으로 강조.
- **노션 스타일 언어 선택기**: 
    - 코드 블록 우측 상단에 커스텀 드롭다운 메뉴 추가.
    - 실시간 언어 검색 필터링 기능 구현.
    - 초기 기본 언어를 **JAVA**로 설정하고 'PLAIN TEXT' 등 불필요한 옵션 정리.
- **에디터 최적화**: 
    - 브라우저 맞춤법 검사(`spellcheck`)를 비활성화하여 코드 내 빨간 밑줄 제거.
    - 코드 블록 배경색 및 폰트 스타일을 시각적으로 최적화(회색톤 배경, Atom One Dark 테마).

## 5. 기타 버그 수정
- `BlockHandle.jsx`에서 발생하던 `nodeSize` 관련 Null 참조 에러 수정.
- 하이라이팅 구현 중 누락되었던 필수 라이브러리(`debounce`, `BlockId` 등) 임포트 복구.
