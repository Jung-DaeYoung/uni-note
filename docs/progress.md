## ✅ 완료된 작업 (Completed)

### 1. 백엔드 (Backend)
- [x] **도메인 모델 설계**: Student, Professor, Course, Note, Post, Enrollment 엔티티 구축.
- [x] **인증 시스템**: Spring Security + JWT 기반 로그인 및 토큰 필터 구현.
- [x] **API 개발**:
    - `AuthController`: 로그인 처리.
    - `DashboardController`: 수강 강의 목록 및 통계 데이터 제공.
    - `NoteController`: 강의별 노트 생성/조회 및 자동 저장 API.
    - `PostController`: 강의별 익명 게시판 CRUD API.
- [x] **DB 연동**: JPA를 활용한 MySQL(H2/MariaDB 등) 매핑 및 Auditing 적용.

### 2. 프론트엔드 (Frontend)
- [x] **환경 구축**: Vite + React + Tailwind CSS v4 디자인 시스템 도입.
- [x] **핵심 페이지 구현**:
    - `LoginPage`: 다크/라이트 모드 레이아웃 및 인증 로직.
    - `DashboardPage`: 카드형 강의 리스트 및 학습 요약 UI.
    - `CourseDetailPage`: 3단 가변 레이아웃(내비-에디터-게시판) 적용.
    - `AnonymousBoardPage`: 익명 게시판 UI 및 검색 기능.
- [x] **기능 통합**: Axios 인터셉터를 통한 JWT 인증 처리 및 실시간 데이터 연동.
- [x] **스마트 기능**: 노트 작성 시 2초 디바운싱 기반 비동기 자동 저장(Auto-save) 구현.
