# Capstone Design 프로젝트 계획서: LMS & 노트 대시보드

## 1. 프로젝트 개요
본 프로젝트는 사용자가 수강 중인 강의를 효율적으로 관리하고, 강의별 맞춤형 필기 노트와 소통을 위한 게시판 기능을 제공하는 학습 관리 시스템(LMS)입니다.

## 2. 주요 기능
### 2.1. 사용자 인증 (Authentication)
- **로그인**: 사용자 계정 정보를 통한 시스템 접근 제어.
- (향후 확장: 회원가입 및 프로필 관리 기능)

### 2.2. 강의 대시보드 (Course Dashboard)
- **수직 나열 인터페이스**: 로그인 후 수강 중인 강의 목록이 세로 방향으로 나열되어 한눈에 확인 가능.
- **강의 선택**: 특정 강의 대시보드 클릭 시 해당 강의의 상세 페이지로 이동.

### 2.3. 강의 상세 페이지 (Course Detail View)
- **필기 전용 노트 템플릿**: 강의별로 최적화된 필기 공간 제공 (텍스트 및 서식 지원).
- **강의별 게시판 (CRUD)**: 
    - 강의 관련 질의응답, 공지사항 등을 관리.
    - 게시글 생성(Create), 조회(Read), 수정(Update), 삭제(Delete) 기능 구현.

## 3. 기술 스택 (Tech Stack)
### Frontend
- **React**: 컴포넌트 기반의 현대적 UI 구축.
- **Axios**: Spring Boot REST API와의 비동기 데이터 통신.
- **React Router**: 싱글 페이지 애플리케이션(SPA) 내 라우팅 관리.
- **CSS Modules / Styled-components**: 대시보드 및 노트 템플릿 스타일링.

### Backend
- **Spring Boot**: 안정적이고 확장 가능한 RESTful API 서버 구축.
- **Spring Data JPA**: MySQL 데이터베이스와의 객체-관계 매핑(ORM).
- **Spring Security**: 사용자 인증 및 API 보안 적용.

### Database
- **MySQL**: 데이터의 정형적 저장 및 관리를 위한 관계형 데이터베이스.

## 4. 데이터베이스 설계 (ERD 예상)
- **User (사용자)**: 아이디, 비밀번호, 이름, 이메일 등.
- **Course (강의)**: 강의명, 교수명, 강의 설명, 시간표 등.
- **Enrollment (수강)**: 사용자-강의 간의 다대다(N:M) 관계 매핑.
- **Note (필기)**: 강의별 작성된 노트 내용, 작성/수정 일시.
- **Post (게시글)**: 강의별 게시판 게시물 정보 (제목, 내용, 작성자, 작성일).

## 5. 개발 단계별 로드맵
1.  **초기 환경 구축**: Spring Boot 프로젝트 세팅(Dependency 추가), React 프로젝트(Create React App/Vite) 초기화.
2.  **데이터베이스 연동**: MySQL 설치 및 JPA 엔티티(Entity) 설계 및 매핑.
3.  **백엔드 API 개발**: JWT 기반 로그인, 강의 목록 조회, 게시판 CRUD API 구현.
4.  **프론트엔드 UI 개발**: 로그인 페이지, 수직형 대시보드, 노트/게시판 레이아웃 구현.
5.  **API 통합 및 테스트**: Axios를 통한 데이터 연동, 기능 테스트 및 버그 수정.
6.  **배포 및 최종 점검**: (필요 시) 서버 배포 및 전체 기능 시연 준비.
